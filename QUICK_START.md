# 🚀 מדריך פריסה מהיר - 5 דקות

## קבצים להעלאה
העלה 5 קבצים:
- app_complete.py
- index_beautiful.html
- requirements.txt
- Procfile
- runtime.txt

## אפשרות 1: Render.com (מומלץ!)
1. render.com → New Web Service
2. Build: pip install -r requirements.txt
3. Start: gunicorn app_complete:app
4. Deploy!

## אפשרות 2: Railway.app
1. railway.app → New Project
2. Upload files
3. Auto-detect Python
4. Deploy!

## אפשרות 3: PythonAnywhere
1. pythonanywhere.com → Sign up
2. Upload files
3. pip install -r requirements.txt
4. Web app → Flask
5. Configure WSGI
6. Reload!
